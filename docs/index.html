user manual.

For recruiters:

  1. Find @AnonymizeIt_bot on Telegram.
  2. Type /beginSession to create a new interview session and secret key.
  3. Send the secret key to the candidate via email.
  4. Once candidate join interview session, type /s ### with ### being your message to the candidate. You are now chatting anonymously with your candidate!

For candidates:

  1. Find @AnonymizeIt_bot on Telegram.
  2. After receiving the secret key from a recruiter, type /joinSession ### with ### being the secret key.
  3. After joining the interview session, type /s ### with ### being your message to the recruiter. You are now chatting anonymously with your recruiter!
  4. Type /endSession to end the current interview session.
